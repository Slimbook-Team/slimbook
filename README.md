Scripts
--
Para ejecutar los scripts;
1. Descargalos
2. Abre una terminal (ctrl+alt+t)
3. Sitúate en la ruta de script:
  ```
  cd <ruta>
  ```
4. Ejecútalo usando bash:
  ```
  bash script.sh
  ```
